3

Account vs. folder

what you do & don't duplicate

The bill

💳 Account / Subscription

Who pays & your usage budget
  • Controls auth, billing, rate limits
  • Shared across all agents
  • One account runs the whole fleet
  • Does not make agents different
Need one per agent? No. Spin up a new account only for hard isolation (e.g. air-gapped finance) — not yet.
The employee

📁 Folder / Workspace

Who the agent actually is
  • Holds identity, memory, tools, permissions
  • Fully isolated per agent
  • This is what makes Felix ≠ Jessica
  • The personality lives here, not the account
Need one per agent? Yes — the folder IS the agent. New hire = new folder.
💡 Scaling lever = where background work bills, not more subscriptions. Start with one account + a handful of folders. When the fleet gets busy, move the cron agents to API-key billing so they never throttle interactive work — optionally one key per agent for per-agent cost tracking & independent revoke. Built API-key-ready: a one-line env change, no restructure.

Autonomy — tiered, earned

same model Sherlock runs on

Tier 1 · Free

Act without asking

  • Read & search
  • Research
  • Draft & plan
  • Log to memory
Tier 2 · Ask first

Approve until proven

  • Send / reply
  • Post to channels
  • Write to systems
  • Schedule / modify
Tier 3 · Never

Hard boundaries

  • Financial transactions
  • Impersonation
  • Deleting w/o instruction
  • New access controls
🎚️ The hand-off dial: as an agent proves a recurring task safe, promote it Tier 2 → standing permission — exactly how SpaceMob, XeWorks & MCG-dashboard became autonomous. This is the biggest productivity unlock, not more agents.
5

Local-first & secured

separation IS the security model

🔒 How privacy is enforced

  • Least privilege — Felix sees finance + sheets; Talia never touches QuickBooks; Jessica never reads candidate files
  • Own guardrail hook per agent, with its own allowlist — a confused agent can only act in its lane
  • Local creds in per-agent config dirs; nothing leaves your machine
  • Audit trail — every action lands in that agent's dated memory log
📂 Folder layout — clone-to-scale
